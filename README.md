# Multi-Tenant SaaS Backend Platform

Production-ready multi-tenant SaaS backend platform built with Java 17 and Spring Boot.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Flyway
- Docker

## Goals
- Support multiple organizations (tenants) in a single system
- Enforce strict tenant data isolation
- Implement JWT-based authentication and role based access control
- Follow production grade backend architecture and practices
